## Start a test

From you vsode terminal change directory to your project root for **lab2** 
__(lab2/cfn-project)__

Under cfn-project you will see the following files
```
cfn-project
├── lambda_functions
├── templates
└── .taskcat.yml

```

From the **cfn-project** dir run you taskcat test

```
taskcat test run
```
